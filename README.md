# Smart Campus Mobile Application for Northern University

A full-stack mobile application developed to digitize university services and improve student access to campus resources through a centralized mobile platform.

## Project Overview

The Smart Campus Mobile Application provides a centralized platform for students and university staff to access campus services and information.

The system includes:

* Authentication and User Management
* Visitor Sign-In Management
* Lost & Found Management
* Library Services
* Announcements and Notifications
* Canteen Services
* Campus Information Services

## Features

### Authentication Module

* User registration and login
* Secure authentication
* User profile management

### Lost & Found Management Module

* Report lost items
* Report found items
* Track item status
* Search and manage item records

### Visitor Sign-In Module

* Visitor registration
* Visitor tracking
* Visit history management

### Library Module

* Library information access
* Resource management

### Announcements Module

* University announcements
* Event notifications

### Canteen Module

* Food and service information

## Installation

### Backend

```bash
cd server
npm install
npm run dev
```

### Mobile Application

```bash
cd mobile
npm install
npx expo start
```
